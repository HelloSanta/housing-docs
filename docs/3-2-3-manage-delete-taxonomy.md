# 刪除分類

#### 【刪除分類路徑操作】

1. 請先點選『管理』
2. 顯示第二層時點選『管理分類』
3. 即可選擇您想要管理的分類群組

![](/_image/manage/taxonomy.png)


#### 【如何刪除分類？請依序以下的步驟完成】
1. 點選分類後方向下的按紐，會顯示編輯與Delete  
  
![](/_image/manage/taxonomy-delete.png)

2. 請點選Delete，會出現二次驗證，請再點選Delete即可  
  
![](/_image/manage/taxonomy-delete-y.png)

以上二個步驟完成後，請點選【儲存】即可
