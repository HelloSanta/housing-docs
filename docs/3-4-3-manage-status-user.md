# 使用者 Status 與角色


#### 【使用者 status 與角色路徑操作】  

1. 請先點選『管理』
2. 顯示第二層時點選『管理使用者』
3. 選擇『使用者』

![](/_image/manage/user.png)


#### 【如何調整使用者 status與角色？請依序以下的步驟完成】  

**● 步驟一**  

請點頁面內『動作』下方的下拉式選單，選擇要執行的動作

![](/_image/manage/user-status.png)


**● 步驟二**  

勾選要執行步驟一的使用者名稱，選擇完成後，按下『套用至選取的項目』，即可完成 Status 或角色的更變。

![](/_image/manage/user-status-c.png)
