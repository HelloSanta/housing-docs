# 使用者取消帳號


### 【使用者取消帳號路徑操作】  

1. 請先點選『管理』
2. 顯示第二層時點選『管理使用者』
3. 選擇『使用者』

![](/_image/manage/user.png)


### 【如何取消使用者帳號？請依序以下的步驟完成】  

#### ● 步驟一  

請點頁面內欲編輯的使用者後方『編輯』按鈕

![](/_image/manage/user-edit.png)

</br>

#### ● 步驟二  

進入使用者頁面後，點選頁面最下方『取消帳號』

</br>

#### ● 步驟三  

選擇欲執行選項，按下『確定』後完成

![](/_image/manage/user-disable.png)
