# 排序分類

### 【排序分類路徑操作】

1. 請先點選『管理』
2. 顯示第二層時點選『管理分類』
3. 即可選擇您想要管理的分類群組

![](/_image/manage/taxonomy.png)

### 【如何排序分類？請依序以下的步驟完成】

1. 列表中名稱前方都會有十字的圖示，請使用滑鼠左鍵，點著十字不放，位置如下圖

![](/_image/manage/taxonomy-sort.png)

2. 確認好位置之後，放開滑鼠左鍵，系統會自動顯示你有尚未儲存的變更，代表有移動到排序囉

![](/_image/manage/taxonomy-sort-y.png)

以上二個步驟完成後，請點選【儲存】即可
