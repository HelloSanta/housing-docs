# 編輯選單

### 【編輯選單路徑操作】  

1. 請先點選『管理』
2. 顯示第二層時點選『管理選單』
3. 選擇『主選單』
4. 編輯的連結頁面會顯示在網頁的 navbar 區塊內

![](/_image/manage/menu.png)

### 【如何編輯選單？請依序以下的步驟完成】   

**● 步驟一**  

在頁面中，點選想要編輯連結名稱後方的『編輯』按鈕，即可進入編輯頁面。

![](/_image/manage/menu-edit.png)

**● 步驟二**  

編輯好選單連結標題、連結等相關欄位後，請務必點選【儲存】即可

![](/_image/manage/menu-edit-page.png)
